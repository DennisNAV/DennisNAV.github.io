Methodology

Tools Used

• Web browser
• Notepad
• Microsoft OneDrive
• Hard drive

Data Collection and Storage

1. Navigated to the Lonergan Research report containing the data required.
2. Examined the data and identified the specific data that was of interest for this study.
3. Data was transferred and converted into a CSV file in the Notepad application.
4. The data was saved to the Microsoft OneDrive cloud and a backup was created on a local hard drive.



Data Manipulation

1. The original data contained within a PDF file was converted into a CSV file.
2. All data was examined to ensure there were no errors or inaccuracies.
3. Only the data required was kept, and all unneeded data was removed from the CSV file. This deleted data included data regarding time spent commuting, eating, working, preparing for day and exercising. Fields relating to location and generation were also removed.
4. The data kept including sleep time, screen time and some qualitative responses were all saved in the CSV file and backed up to the cloud as well as saved locally.
5. Upon inspection it was determined that further manipulation of data was unneeded, as it was already stored clearly under specific fields. No data needed to be combined or separated.
6. The data structure and file naming conventions used are as shown below.
